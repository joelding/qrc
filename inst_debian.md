
# INSTALL DEBIAN
Installing Debian as development host.
## Install
```
$ su
# apt-get install vim git gitk ctags cscope build-essential fonts-inconsolata curl
```
## Setup hotkeys
|Hotkey|Command|
|--|--|
| `<Alt>t`|lxterminal|

## Format a new disk
```
cfdisk
```
