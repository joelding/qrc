#### i2c tools for linux

* originally: lm-sensors package

`````
$ git clone git://git.kernel.org/pub/scm/utils/i2c-tools/i2c-tools.git
`````
or download from [link](https://www.kernel.org/pub/software/utils/i2c-tools/)

* compile by
````````
$ CC=arm-hisiv500-linux-gcc make
````````
